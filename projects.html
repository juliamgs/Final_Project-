My Projects

A collection of projects showcasing my approach to user-centered design and development.

Stellantis - AI Predictive Maintenance UX System

Role: UX Designer

Timeline: Aug 2024 - Dec 2024

Partnered with Stellantis to design an AI-driven predictive maintenance UX system for the 2030 automotive market. This innovative project focused on creating intuitive interfaces that help drivers understand and respond to vehicle maintenance needs proactively, leveraging artificial intelligence to predict issues before they become problems.

Key Achievements:

  • Designed AI-powered predictive maintenance alerts and notification system
  • Created user-friendly dashboard interfaces for vehicle health monitoring
  • Conducted extensive automotive UX research with diverse driver demographics
  • Developed future-focused design solutions for 2030 automotive market
  • Collaborated with Stellantis engineering teams on feasibility and implementation

Tools & Methods: Figma, User Research, Journey Mapping, Prototyping, Usability Testing

Skymood - In-Flight Entertainment System

Role: UX Designer & Researcher

Timeline: Aug 2024 - Dec 2024

Created a transformative in-flight entertainment system that personalizes content based on anxiety assessments to improve passenger experience. This empathetic design approach considers the emotional states of passengers and adapts entertainment options to help them feel more comfortable during flights.

Key Achievements:

  • Integrated anxiety assessment tools with content personalization algorithms
  • Designed empathetic user interfaces for diverse passenger needs
  • Conducted in-depth passenger experience research and interviews
  • Created high-fidelity prototypes with interactive entertainment flows
  • Developed accessibility features for passengers with various needs

Tools & Methods: Figma, User Research, Persona Development, Prototyping, A/B Testing

Sahiyo - Global Network Map Redesign

Role: Remote UX Designer

Timeline: Sep 2024 - Apr 2025

Leading the redesign of an interactive Global Network Map for Sahiyo, a nonprofit organization working to end FGM/C (Female Genital Mutilation/Cutting). The project focuses on amplifying survivor stories and improving the visibility and accessibility of the organization's global impact.

Key Achievements:

  • Redesigned interactive map visualization for better data storytelling
  • Created sensitive, trauma-informed design approaches for survivor stories
  • Improved accessibility features for international audiences
  • Developed responsive design for mobile and desktop experiences
  • Collaborated with nonprofit stakeholders on mission-aligned design

Tools & Methods: Figma, Adobe XD, Data Visualization, User Research, Stakeholder Interviews

UMSI Career Development Website

Role: UX Designer & Front-End Developer

Timeline: Fall 2024

Designed and developed a mobile-first website to help UMSI students navigate career development resources. The project emphasized accessibility, clear information architecture, and responsive design to ensure all students can easily find and utilize career services.

Key Achievements:

  • Implemented mobile-first responsive design with CSS Grid
  • Achieved WCAG AA accessibility compliance with comprehensive testing
  • Created intuitive navigation with skip-to-content functionality
  • Designed according to University of Michigan brand guidelines
  • Improved information architecture based on UX research feedback

Tools & Methods: HTML5, CSS3, Figma, WAVE, Axe Accessibility Testing

Roos Roast Coffee - E-commerce Redesign
